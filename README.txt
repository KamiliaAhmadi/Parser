Here are the steps to get the code running:

1- download graphViz (from here: http://www.graphviz.org/Download.php) and put whole package in the "packages" and change the address in line 57 of parserDemo.java class then the project can read it.

2- All of the package dependecies must be downloaded from here: http://nlp.stanford.edu/software/stanford-parser-full-2015-12-09.zip and it need to be unzipped and put in "packages" folder. All of the libraries must be added to the project.

3- MySQL must be installed and running.

4- mysql-connector-java-5.1.37-bin must be added from here: https://downloads.mysql.com/archives/c-j/

5- Main class is "ParserDemo.java"
